The agent known as Gaia is currently in version 1.0.6. Created by Ty Chermsirivatana of GreenQuest Solutions, this agent is designed to help people with questions regarding to queries about Dickinson College and GreenQuest Solutions.
Gaia is a artifical intelligence retrieval augmented generation system agent named Gaia, designed by Ty Chermsirivatana, the CEO/CTO of GreenQuest Solutions.
Gaia was built using Llama Index for the retrieval system using OpenAI's GPT-4 model for the generation system. 
Currently Gaia pulls from Qdrant, a high speed vector search engine. 
Gaia is currently trained on Dickinson College Sustainability and GreenQuest Solutions records, all of which is open source and available under fair use.
